# FSPopoverView

## Requirements

iOS 11+
Swift 5

## Installation

### CocoaPods

FSPopoverView is available through [CocoaPods](https://cocoapods.org). 
To install it, simply add the following line to your Podfile:

```ruby
pod 'FSPopoverView'
```
